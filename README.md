# Site Web d'Anniversaire pour Jean

Une application Flask interactive pour souhaiter joyeux anniversaire à Jean avec un compte à rebours initial, des feux d'artifice et une carte à télécharger.

## Fonctionnalités

- **Compte à rebours initial de 5 secondes** avec animation avant l'affichage du site
- **Son de joyeux anniversaire** joué automatiquement après les 5 secondes
- **Compte à rebours principal** jusqu'à la date d'anniversaire de Jean
- **Feux d'artifice interactifs** avec effets visuels réalistes
- **Message d'anniversaire animé** avec gâteau et bougie
- **Bouton pour télécharger une carte d'anniversaire** en format PDF
- **Confettis colorés et décorations flottantes**
- **Musique d'anniversaire**
- **Interface responsive et moderne**
